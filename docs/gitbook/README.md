# Introduction

DeSNet is a decentralized social network protocol on Aptos. Every profile is an Object NFT, every profile spawns its own fungible token, and every social action — posts, likes, replies, quotes, presses, syncs — is an on-chain primitive.

## Key Features

- **No Centralized Backend**: No off-chain database, no protocol fees on swaps.
- **Identity as Assets**: Profiles (PID) are transferable Object NFTs.
- **Social Economy**: Every profile has its own $TOKEN and AMM pool.
- **The Seven Verbs**: All social actions (Mint, Spark, Voice, Echo, Remix, Press, Sync) are on-chain primitives.
- **MEV-Safe Economics**: Protocol buybacks use a commit-reveal mechanism.

## Status

**v0.3.3** is currently live on Aptos mainnet. The protocol consists of 18 Move modules and has been audited by a 6-LLM panel.
